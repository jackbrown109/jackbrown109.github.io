VR Project

This project's aim was to produce a game using VR set in Park Campus' Waterworth Building.

My task was to produce a procrastination mini-game where the player will find bottles of beer around the environment.

The player can then interact with the bottle by picking it up.

However, if they tilt the bottle, the liquid inside will pour out (represented by a particle system), causing the player to get 'Drunk'. This activates a blur shader which fades if they stop drinking.
